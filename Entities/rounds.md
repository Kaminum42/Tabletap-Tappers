## 字段

- `round_id`
    - 数据类型: `number`
    - 描述: 对局记录 ID
    - 约束: 整数, 0 ~ 2^32-1
    - 是否必须: 是
    - 示例: `1`

- `associated_game`
    - 数据类型: `number`
    - 描述: 关联的游戏 ID
    - 约束: 整数, 0 ~ 2^32-1
    - 是否必须: 是
    - 示例: `1`

- `players`
    - 数据类型: `Array`
    - 描述: 参与本局的玩家
    - 约束: --
    - 是否必须: 是
    - 示例: --

    - `user_id`
        - 数据类型: `number`
        - 描述: 用户 ID
        - 约束: 整数, 0 ~ 2^32-1
        - 是否必须: 是
        - 示例: `10000`

- `create_time`
    - 数据类型: `number`
    - 描述: 对局记录创建时间
    - 约束: 时间戳(ms), 整数, 0 ~ 2^64-1
    - 是否必须: 否
    - 示例: `1704038400000`

- `scores`
    - 数据类型: `Array`
    - 描述: 每个玩家的得分
    - 约束: --
    - 是否必须: 是
    - 示例: --

    - `score`
        - 数据类型: `number`
        - 描述: 得分
        - 约束: 整数, 0 ~ 2^32-1
        - 是否必须: 是
        - 示例: `10`

- `details`
    - 数据类型: `string`
    - 描述: 其他详细信息
    - 约束: JSON 格式
    - 是否必须: 否
    - 示例: `{"key": "value"}`
